# nmxu-pr
SSR - REST API (Nuxt v2, Axios &amp; Vuex)

### Установка / Install

Выполнить в командной строке: / Execute in your command line:

* Git: <br/>
$ git clone https://github.com/CoreLabx64/nxmu-pr.git

* Yarn: <br/>
$ yarn install

* Npm: <br/>
$ npm install

### Запуск / Run

Выполнить в командной строке: / Execute in your command line:

* Yarn: <br/>
$ yarn run build <br/>
$ yarn run start

* Npm: <br/>
$ npm run build <br/>
$ npm run start
